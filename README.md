# StatsGlobal — Project Zomboid Mod

> Lightweight real-time HUD with progress bars for the stats that matter most.
> Compatible with **Singleplayer** and **Multiplayer Hosted**.

---

## 🇺🇸 English

### What it does

StatsGlobal adds a small, draggable HUD panel to your screen showing 5 key stats as live progress bars:

| Stat | Description |
|------|-------------|
| 🧟 Zombies | Total zombie kills |
| 🍖 Hunger | Current hunger level with color-coded severity |
| 💧 Thirst | Current thirst level with color-coded severity |
| 🥾 Distance | Kilometers and meters walked |
| 🌡️ Temperature | Body temperature in Celsius |

### Features

- Updates in real time every tick
- Color changes based on severity (green → yellow → orange → red)
- Labels reflect actual game moodle thresholds (Peckish, Hungry, Starving, etc.)
- Draggable panel — position it anywhere on screen
- No dependencies, no server-side files required
- Works on hosted multiplayer — each player sees their own stats

### Installation

1. Copy the `StatsGlobal` folder to:
   ```
   C:\Users\[YourName]\Zomboid\mods\
   ```
2. Launch Project Zomboid
3. Go to **Mods** → enable **StatsGlobal**
4. Start or join a game

### Compatibility

- Project Zomboid Build 41 / 42
- Singleplayer ✅
- Multiplayer Hosted ✅
- Dedicated Server ✅ (client-side only, no server mod needed)

## 🇧🇷 Português

### O que faz

O StatsGlobal adiciona um painel HUD pequeno e arrastável na tela com 5 stats importantes em barras de progresso em tempo real:

| Stat | Descrição |
|------|-----------|
| 🧟 Zumbis | Total de zumbis mortos |
| 🍖 Fome | Nível de fome com cores por severidade |
| 💧 Sede | Nível de sede com cores por severidade |
| 🥾 Distância | Quilômetros e metros andados |
| 🌡️ Temperatura | Temperatura corporal em Celsius |

### Funcionalidades

- Atualiza em tempo real a cada tick
- Cores mudam conforme a gravidade (verde → amarelo → laranja → vermelho)
- Labels refletem os thresholds reais do jogo (Com fome, Faminto, Morrendo de fome, etc.)
- Painel arrastável — posicione onde quiser na tela
- Sem dependências, sem arquivos no servidor
- Funciona em multiplayer hosted — cada jogador vê seus próprios stats

### Instalação

1. Copie a pasta `StatsGlobal` para:
   ```
   C:\Users\[SeuNome]\Zomboid\mods\
   ```
2. Abra o Project Zomboid
3. Vá em **Mods** → ative o **StatsGlobal**
4. Entre em uma partida

### Compatibilidade

- Project Zomboid Build 41 / 42
- Single Player ✅
- Multiplayer Hosted ✅
- Servidor Dedicado ✅ (só client-side, não precisa instalar no servidor)
